Standard

This game mode got inspired by a rhythm game named Osu! Tatakae! Ouendan which have a Japanese-only version released in 2005. Besides, it has a fairly simple gameplay where players tap on circles, sliders, and spinners appear on the screen in time with music that play in the background. Moreover, it emphasises precision and rhythm with a health bar that reduced over time and can be recovered by clicking on those objects show on screen accurately. The scoring system rewards accuracy, combo, and timing with higher scores for well-timed hits and maintained combos.

Taiko

This game mode got inspired by a rhythm game named Taiko No Tatsujin which first released in 2001. Besides, it has a fairly simple gameplay where players hit notes, sliders, and spinners in time with the music using a taiko drum (or a keyboard). Notes come in red and blue which require quick fingers switching ability, and special notes like drumrolls and dendens that add variety to the gameplay. Moreover, the health bar must be at least 50% full to pass a song, and scoring is based on accuracy, combo, and timing with bonuses during Kiai Time.

Catch

This game mode got inspired by a hidden mode unlocked from a game named rhythm-it (or ruv-it) which first explored in 2001. Besides, it has a fairly simple gameplay where players control a character that catches falling fruit in tune with the beat. The game features various types of fruits, fruit trails, bananas, and hyperfruits, each with unique scoring and gameplay mechanics. Moreover, the zoom size of the playfield is based on Circle Size (CS) settings, and the catcher can move left and right to catch fruits. Scoring is based on the number of fruits caught, combo, and accuracy, with a health bar that depletes on misses.

Mania

This game mode got inspired by a rhythm game named Beatmania IIDX which first released in 1999. Besides, it has a fairly simple gameplay where players hit notes that falling down from the top of the machine-like box. It supports various key configurations, mimicking different rhythm games like Guitar Hero or Dance Dance Revolution. Moreover, it features hold notes and a unique scoring system that combines accuracy and combo, with a health bar that only reduced hit player misses a key click or more.
